Tutorial OpenVPN Connect
Untuk menggunakan OpenVPN di perangkat Android, ikuti langkah-langkah dibawah ini.
Unduh dan Install OpenVPN Connect
Anda dapat mengunduh aplikasi OpenVPN Connect dari Google Play Store dan menginstalnya di perangkat Android Anda.
Unduh File Konfigurasi VPN
OpenVPN Connect tidak akan berjalan tanpa File Configurasi VPN. Untuk mendapatkan file konfigurasi VPN, kita harus mendownload dari situs penyedia layanan file konfigurasi VPN. Biasanya, file ini berupa file dengan ekstensi .ovpn.
Berikut ini situs Gratisan Penyedia File Konfigurasi VPN :
✓ Situs OMGVPN
✓ Situs VPNGATE
Jika sudah mengunduh File Konfigurasi, Impor File Konfigurasi ke Aplikasi OpenVPN di perangkat Android Anda. Caranya adalah:
✓ Klik hasil unduhan file konfigurasi open Vpn, Maka anda akan diarahkan ke aplikasi Open Vpn.
✓ Pada pemberitahuan "Import .ovpn profile ?" , klik saja OK.
✓ Lalu Klik "Connect"
✓ Selesai.
Note : Jika menggunakan user dan password, maka wajib mengisi user dan password tersebut.
Cara Download File Konfigurasi Open Vpn di Omgvpn.com
✓ Buka situs omgvpn.com
✓ Klik Menu lalu klik pada VPN Gate Full
✓ Setelah jendela terbuka, geser kebawah hingga menemukan daftar Vpn Gate Server seperti Gambar dibawah ini.
✓ Lalu cari dan pilih Server Negara yang diinginkan.
✓ setelah menemukan server negara yang diinginkan, klik Connect dibawah gambar bendera server tersebut seperti gambar diatas.
✓ Setelah jendela baru terbuka, geser kebawah, maka kamu akan menemukan username dan password yang akan digunakan, biasanya username dan password = vpn. seperti gambar dibawah ini.
✓ Geser lagi kebawah hingga menemukan tulisan "Click To Download" untuk mendownload file config seperti gambar dibawah ini.
✓ Pilih salah satu file config yang diinginkan. lalu klik "Click To Download", maka file akan otomatis terdownload.
Selesai !
