Tutorial Tun2TAP
Untuk memasang IP Proxy menggunakan aplikasi Tun2TAP, ikuti langkah-langkah dibawah ini.
Unduh dan Install Tun2TAP
Anda dapat mengunduh aplikasi Tun2TAP dari Google Play Store dan menginstalnya di perangkat Android Anda.
Buka Aplikasi Tun2TAP
Setelah Anda menginstal aplikasi, buka Tun2TAP di perangkat Anda.
Konfigurasi Proxy
Di layar utama Tun2TAP, ikuti langkah-langkah berikut:
✓ "Copy" Ip dan Port Proxy yang ingin digunakan.
✓ Lalu "paste" kan pada kolom yang telah disediakan.
✓ Lalu klik pada tombol "Connect", seperti gambar dibawah.
✓ Selesai.
